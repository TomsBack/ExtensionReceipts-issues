# Extension Receipts Privacy Policy

Last updated: June 10, 2026

Extension Receipts does not collect, sell, share, transmit, or store personal data outside your browser.

Extension Receipts runs locally in Chrome to explain installed extension permissions, host access, enabled state, and risk signals in plain English.

## Data Stored Locally

Extension Receipts stores local UI settings and previous enabled-state snapshots in Chrome extension storage. This data stays in your browser unless you use browser sync features controlled by Chrome.

## Extension Metadata

Extension Receipts uses Chrome's management API to read installed extension names, descriptions, enabled states, permissions, host permissions, and related metadata so it can explain them locally. This metadata is not transmitted.

## Data Collection

Extension Receipts does not use analytics, advertising SDKs, external tracking, remote logging, or external servers. It does not transmit installed-extension metadata, browsing history, page content, or settings to the developer or any third party.

## Permissions

Extension Receipts requests browser permissions only to provide local extension-permission analysis: management access for installed extension metadata and storage access for local settings/snapshots.
